# enarx-keepmgr

This crate provides the `enarx-keepmgr` executable which creates and
coordinates enarx-keepldr instances

## Build

    $ git clone https://github.com/enarx/enarx-keepmgr
    $ cd enarx-keepmgr
    $ cargo build

## Run Tests

    $ cargo run enarx-keepmgr-tester
